# Piedra, Papel o Tijera – Proyecto Final

Este proyecto contiene dos versiones del juego Piedra, Papel o Tijera:  
1. Una versión hecha en Python  
2. Una versión hecha en RAPTOR  

El propósito del proyecto es mostrar una implementación sencilla del juego usando dos herramientas distintas.

# Versión en Python

La versión en Python se ejecuta en la terminal.  
El jugador elige una opción (piedra, papel o tijera) y el programa genera aleatoriamente la jugada del NPC.  
Luego se muestra quién gana o si hay empate.

### Cómo ejecutarlo

python3 Game.py o python Game.py (por si da error uno)

# Versión en RAPTOR

El archivo RAPTOR contiene el mismo juego pero en forma de diagrama de flujo.  
Permite seleccionar una jugada y el programa genera otra aleatoria del NPC, mostrando el resultado final.

Para usarlo solo debes abrir el archivo `.rap` con RAPTOR y ejecutarlo.


## Cómo se juega

El juego funciona igual en ambas versiones:

- Piedra vence a tijera  
- Tijera vence a papel  
- Papel vence a piedra  
- Si ambos eligen lo mismo, es empate  
- En ambas versiones se puede elegir jugar de nuevo o salir (aunque en raptor se sale cada partida que terminas)
