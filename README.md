# Piedra, Papel o Tijera – Proyecto Integrador

## Descripción del proyecto

Este proyecto forma parte del **Proyecto Integrador** de la asignatura y consiste en el desarrollo del juego **Piedra, Papel o Tijera**, implementado en dos versiones:

1. Una versión desarrollada en **Python**
2. Una versión representada mediante un **diagrama de flujo en RAPTOR**

El proyecto busca demostrar cómo incluso un software sencillo puede representar el **impacto de las nuevas tecnologías en la sociedad**, especialmente en el entretenimiento digital y la automatización de decisiones.

---

## Objetivo del proyecto

Analizar el impacto de la tecnología en la sociedad a través del desarrollo de un software interactivo que simula decisiones humanas, permitiendo comprender el funcionamiento básico de los sistemas digitales y su aplicación en la vida cotidiana.

---

## Relación con el impacto social de la tecnología

* La tecnología facilita el **entretenimiento digital accesible**.
* Automatiza procesos similares a la **toma de decisiones humanas**.
* Los videojuegos son una forma común de **interacción tecnológica** en la sociedad.
* Este proyecto refleja cómo la programación permite crear sistemas interactivos funcionales.

* Diagramas de flujo explicativos

---

## Versión en Python

La versión en Python se ejecuta desde la terminal.
El jugador elige una opción (piedra, papel o tijera) y el programa genera de forma aleatoria la jugada de la computadora.
Posteriormente, el sistema compara ambas elecciones y muestra el resultado de la partida.

### Cómo ejecutarlo

```bash
python3 piedra_papel_tijera.py
```

o

```bash
python piedra_papel_tijera.py
```

---

## Versión en RAPTOR

El archivo RAPTOR representa el mismo juego mediante un diagrama de flujo.
Permite visualizar de forma gráfica la lógica del programa y la toma de decisiones automatizadas.

Para utilizarlo:

* Abrir el archivo `.rap` con RAPTOR
* Ejecutar el diagrama

---

## ¿Cómo se juega?

El funcionamiento del juego es el mismo en ambas versiones:

* Piedra vence a tijera
* Tijera vence a papel
* Papel vence a piedra
* Si ambos eligen la misma opción, el resultado es empate
* El jugador puede decidir si desea volver a jugar o salir del programa
